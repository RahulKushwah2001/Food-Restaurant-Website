## 🌟 Overview

A modern and responsive **Food Restaurant Website** built using HTML, CSS, JavaScript, and Bootstrap.
This project features an interactive UI, smooth animations, and a functional **Add-to-Cart system** for a seamless food ordering experience.

---

## 🚀 Features

* 🛒 Add to Cart functionality
* 🎨 Modern and attractive UI design
* 📱 Fully responsive (mobile-friendly)
* ⚡ Smooth animations and transitions
* 🍔 Interactive food menu section
* 🧾 Cart page with dynamic updates
* 💻 Clean and structured code

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/RahulKushwah2001/Food-Restaurant-Website.git
```

2. Open the project folder
3. Run `main.htm` in your browser
